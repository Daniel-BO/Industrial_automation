Características principales del programa:
Control completo de botoneras:

Botones SUBIR y BAJAR en cada piso

Botonera interior con selección de piso

Lógica inteligente para solicitudes

Seguridad reforzada:

Interlocks en puertas de ambos pisos

Paro de emergencia con prioridad máxima

Tiempo máximo de movimiento por seguridad

Detección de límites de carrera

Control de polipasto:

Salidas separadas para subir y bajar

Protección contra sobrepaso de límites

Verificación de puertas antes del movimiento

Indicadores visuales:

LEDs indicadores de piso actual

Luz de cabina automática

Alarma para condiciones anormales

Diagnóstico:

Mensajes seriales para seguimiento

Estado del sistema en todo momento

Este programa está optimizado para un elevador de carga con polipasto, proporcionando control preciso y máxima seguridad.
